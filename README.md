# Sistem Akademik SMAN 8 Palembang

Aplikasi Sistem Informasi Akademik Sekolah SMAN 8 Palembang berbasis Java Spring Boot + Thymeleaf.

## Teknologi
- Java 17
- Spring Boot 3.4.3
- JPA / Hibernate
- H2 / PostgreSQL
- Docker

## Cara Menjalankan Secara Lokal
```bash
# Menggunakan Maven wrapper
./mvnw spring-boot:run
```
